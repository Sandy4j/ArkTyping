# ArkTyping — Typeknights

Deskripsi Singkat
-----------------
Typeknights (ArkTyping) adalah game action tower-defense berbasis mekanik pengetikan (typing). Pemain memanggil hero dan mengaktifkan skill mereka dengan mengetik cepat dan akurat. Setiap huruf yang diketik ibarat mantra, setiap kata merupakan serangan. Bertahanlah dari gelombang monster Abyss dalam nuansa dark fantasy.

Fitur Utama
-----------
- Mekanik pengetikan sebagai inti permainan: memanggil hero dan mengaktifkan skill melalui teks.
- Penempatan hero pada tile altar (grid 1–9) untuk strategi pertahanan.
- Sistem sumber daya: Summon Point dikumpulkan dari musuh yang dikalahkan.
- Hero unik dengan jangkauan, peran, dan biaya summon berbeda.
- Wave-based progression: musuh dan pola serangan bertambah menantang setiap level.
- Nuansa Arknights-inspired dengan modifikasi gameplay berbasis typing.

Core Gameplay Loop
------------------
1. Mulai level.
2. Pilih tile altar (1–9) untuk menempatkan hero.
3. Kelola Summon Point (resource) untuk memanggil hero atau menggunakan kemampuan.
4. Ketik command untuk:
   - Menyummon hero (ketik nama hero, mis. `kaelio`, `silvanna` lalu Enter)
   - Mengaktifkan skill (ketik nama skill, mis. `overload burst`, `lunar blessing` lalu Enter)
5. Hadapi wave musuh; hero menyerang otomatis sesuai jangkauan.
6. Gunakan skill dengan timing yang pas (ada cooldown).
7. Selesaikan seluruh wave → lanjut ke level berikutnya.

Kontrol & Input
---------------
Aksi | Tombol / Input | Keterangan
---|---:|---
Pilih Tile Altar | 1–9 | Menentukan posisi hero akan dipanggil.
Command Summon Hero | Ketik nama hero (mis. `kaelio`, `silvanna`) lalu Enter | Men-summon hero di altar yang dipilih (membutuhkan Summon Point).
Command Skill Hero | Ketik nama skill (mis. `overload burst`, `lunar blessing`) lalu Enter | Mengaktifkan skill aktif hero (jika cooldown selesai).
Konfirmasi Command | Enter | Menjalankan command yang diketik.
Pause / Resume Game | ESC | Menjeda atau melanjutkan permainan.
Hapus Input | Backspace | Menghapus command yang sedang diketik.
Menarik Hero (Retreat) | Pilih tile yang akan diretrit, lalu ketik `retreat` | Menarik hero dari field dan mengosongi slot tile altar.
